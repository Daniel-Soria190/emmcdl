## Download & build & install (need a "g++" compiler)
```bash
git clone https://github.com/Zalexanninev15/emmcdl && cd emmcdl && aclocal && autoconf && automake --add-missing && ./configure && make && sudo cp emmcdl /usr/bin/emmcdl && cd .. && rm -rf emmcdl
```
